Help & Support

Welcome to the Student Management System Help Page

This system allows superusers to manage students, subjects, and results efficiently. Below are some tips to get started:

  • Add a Student: Use the "Add Student" button to create a new student profile with their details and credentials.
  • Manage Subjects: Add or edit subjects using the "Add Subject" button. Each subject requires a name and code.
  • Add Results: For each student, you can add results by selecting a subject, entering a score, semester, year, and optional remarks. Grades are calculated automatically based on the score.
  • Filter Results: Use the semester filter to view results for a specific semester.
  • View Charts: Click "View Results" for any student to see a doughnut chart of their grade distribution.
  • Logout: Use the logout link in the navigation bar to securely exit the system.

If you encounter any issues, please contact the system administrator.
